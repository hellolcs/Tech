# RaspberryPi

## Update Raspbian
```
$apt-get update
$apt-get upgrade
```
## Install Korean ibus and font
```
$apt-get install ibus
$apt-get install ibus-hangul
$apt-get install fonts-unfonts-core
```


## apt-get command
* package view
```
$apt-cache show xxx
```

* package search
```
$apt-cache search xxx
```

* install
```
$apt-get install
```

* remove
```
$apt-get --purge remove xxxx
```

* package list
```
$dpkg -l
```
